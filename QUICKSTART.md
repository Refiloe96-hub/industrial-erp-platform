# Quick Start Guide

## Industrial ERP Platform - Phase 11

### 🚀 Get Running in 3 Steps

#### 1. Install Dependencies
```bash
npm install
```

#### 2. Start Development Server
```bash
npm run dev
```

The app will open at `http://localhost:5173`

#### 3. First Time Setup
- Enter your business details
- The app will work **offline-first**
- Data is stored locally in IndexedDB
- Changes sync automatically when online

---

## 📱 Installing as PWA

### Desktop (Chrome/Edge)
1. Open the app in browser
2. Look for install icon in address bar
3. Click "Install"

### Mobile (Android/iOS)
1. Open in Chrome/Safari
2. Tap menu (⋮)
3. Select "Add to Home Screen"

## 🧭 Module Overview

### 1. PocketBooks 💰
Track income, expenses, and cash flow

**Quick Actions:**
- Record transaction
- View cash flow report

### 2. PoolStock 📦

**Quick Actions:**

### 3. SmartShift ⚙️
Schedule production and optimize machines

Form business syndicates and access capital

**Quick Actions:**
- Create/join syndicate

### 5. PocketWallet 💳
Process B2B payments

**Quick Actions:**

## 🔄 Offline Mode

The app works completely offline:
- All data stored locally
- Changes queued for sync
- Auto-sync when back online
- No data loss

Look for the offline banner at top when disconnected.

## 🆘 Troubleshooting

### App won't load
- Clear browser cache
- Check browser console for errors
- Try incognito/private mode

### Data not syncing
- Check internet connection
- Look for sync status in settings
- Force sync from settings menu

### Performance issues
- Clear old synced data
- Check IndexedDB storage
- Update to latest browser
